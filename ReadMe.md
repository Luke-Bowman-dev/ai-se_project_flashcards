# Flashcard App

This repository is a project for the Software Developer Course at TripleTen. It is intended to be a web application that allows the user to create and view Flashcards for the purpose of learning whatever subject they desire.

## Features

This website contains decks of flashcards, with their titles on them and they are color coded. You can click on a deck to see all the cards inside. The cards have their questions on the visible side, and when you click the flip button it shows the answer. You can also enter practice mode which only shows one card at a time for you to practice with. The delete button can delete a deck or a card within a deck. There is also, now, a mobile-friendly view.

## Technologies Used

HTML, CSS, and JavaScript in VS Code
